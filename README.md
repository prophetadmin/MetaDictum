# MetaDictum

MetaDictum is an artifact-governed framework for multi-session AI-assisted
project execution, designed to keep project continuity in repository artifacts
instead of fragile chat context.

This repository provides the documentation and reusable template that implement
that workflow.

The operational scaffold lives in `template/project/`.

## Repository Contents

- `template/project/` contains the reusable project template and execution
  surface.
- `docs/framework/` contains human-facing framework documentation.

## Template Layers

- `01_GENESIS/` for raw ideas and seed creation
- `02_EXODUS/` for implementation work
- `03_LEVITICUS/` for prompts, schemas, and runtime governance artifacts
- `04_DEUTERONOMY/` for optional distilled canon
- `05_NUMBERS/` for optional session and history artifacts
